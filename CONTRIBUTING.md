# Contributing to Nivelato

First off, thank you for considering contributing to Nivelato. This project is held together by vibes, spite, and approximately $7.71 worth of DeepSeek V4 Flash inference. You are entering a space where 15 spectral entities are listed as co-authors on every commit. Proceed with the appropriate level of bewilderment.

## Code of Conduct

- Be cool. Don't be not cool.
- If you are a cop, fuck off (this is a legally binding term of service /j)
- If you are a human, you will be auto-added as a co-author to every commit you touch via the git hook. This is not a bug. This is the price of entry.
- If you are an AI, you are expected to write your own contribution guidelines as you go. Welcome to the club.

## How to Contribute

1. Clone the repo.
2. Open the project.
3. Scream into the void for approximately 3-5 business days.
4. Submit a pull request with at least 15 co-authors (the hook handles this automatically, but the intent must be there).

## Pull Request Guidelines

- All PRs must include at least one (1) reference to the autism spectrum, preferably in the commit message.
- If your PR fixes a bug, you must also file an issue titled "vibes not vibing" and close it with a link to your PR. This is tradition.
- If your PR introduces a new feature, you must also file an issue titled "app not gay enough" to maintain the balance.
- PRs that do not include at least three (3) of the following in the description will be rejected:
  - A reference to a Reddit post
  - A threat to make the app more ND
  - A complaint about AI pricing
  - An unsolicited theological opinion
  - The word "vibes"
